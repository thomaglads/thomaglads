### [Credit Default Risk Analysis](https://github.com/thomaglads/credit-default-risk-analysis)
Predicted loan default probability across 300,000+ applicants by merging 10 relational data sources and engineering 20+ domain-specific features (debt-to-income ratio, payment behavior). Progressed from a Logistic Regression baseline (AUC 0.51) to a tuned LightGBM model (AUC 0.77). Deployed as a Streamlit app for live credit officer use.
`Python` `LightGBM` `SHAP` `Feature Engineering` `Streamlit` `Class Imbalance`

### [Olist E-Commerce Analysis](https://github.com/thomaglads/Olist-E-commerce-Analysis)
Transformed 100,000+ orders across 9 relational tables into a full BI platform using MySQL. Implemented RFM customer segmentation and cohort retention analysis directly in SQL. Built a correlation analysis proving the impact of shipping delays on review scores, visualized in Power BI for marketing and logistics teams.
`MySQL` `SQL` `RFM Analysis` `Cohort Analysis` `Power BI` `ETL`

### [Product Conversion Funnel Analysis](https://github.com/thomaglads/Product-Conversion-Funnel-Analysis)
Analyzed 42 million e-commerce events to identify where users dropped off in the sales journey. Used memory-efficient columnar loading in Python to handle the scale. Discovered an 89% View-to-Cart drop-off and a strong 60% Cart-to-Purchase rate — recommending CRO investment over ad spend increases.
`Python` `Pandas` `Big Data Optimization` `Power BI` `Conversion Rate Optimization`

Designed and analyzed a controlled experiment comparing personalized vs. generic messaging strategies. Used statistical hypothesis testing (Z-test, p-value analysis) to validate a statistically significant lift in engagement — moving beyond vanity metrics to prove real ROI.
`Python` `A/B Testing` `Hypothesis Testing` `Statistical Significance` `SciPy`
